## 📌 Sobre o Projeto

O **Autel** é uma plataforma web para gerenciamento de um hotel de animais domésticos. O sistema permite que tutores realizem reservas de hospedagem para seus pets, acompanhem o período de estadia e contratem serviços adicionais — tudo de forma simples e online.

> Projeto integrador desenvolvido para a disciplina de PI — 3º Ano A.

---

## ✨ Funcionalidades

### 👤 Usuário
- Cadastro e autenticação de usuários
- Cadastro de pets (espécie, raça, porte, comportamento, saúde)
- Atualização de dados cadastrais

### 🏨 Hotelaria
- Realização de reservas de hospedagem
- Visualização de vagas disponíveis por período
- Edição e cancelamento de reservas
- Cálculo automático do valor da hospedagem por diária
- Cálculo de multa por cancelamento fora do prazo
- Contratação de serviços adicionais

### 🛠️ Administração
- Consulta de todos os clientes e pets cadastrados
- Visualização de hospedagens ativas
- Gestão de check-in e check-out

### 🌐 Institucional
- Página inicial com apresentação do Autel
- Página "Quem Somos"
- Página de contatos
- Depoimentos de clientes
